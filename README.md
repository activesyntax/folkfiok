# Folk Fiók – Heritage Education Website

![Folk Fiók – Heritage Education Website Screenshot](/img/folkfiok-screenshot.png)

## Overview

**Folk Fiók** is a heritage education website created for an initiative dedicated to preserving and presenting **Hungarian folk traditions and rural cultural heritage**. The platform serves as an informational and educational hub, combining cultural content with practical program and product presentation.

The project focuses on clear communication, structured content, and accessibility for educators, students, and the general public.

---

## Project Description

The website introduces the mission of the Folk Fiók initiative, presents related books and educational materials, and provides detailed information about **interactive school programs**. Content is organized through a clear and intuitive information architecture, allowing visitors to easily navigate between educational resources, events, and multimedia materials.

The design follows a **clean, structured layout** enriched with **folk-inspired visual elements**, reflecting traditional aesthetics while maintaining modern web usability standards. The site includes dedicated product pages, event listings, and integrated video content to support educational engagement.

---

## Features

- Presentation of cultural and educational mission  
- Book and educational material showcases  
- Interactive school program descriptions

- Clear navigation and intuitive information structure  
- Folk-inspired visual design combined with modern layout principles
